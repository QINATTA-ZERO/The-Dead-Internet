Domain Provisioning

Aether allows instant registration of .psx domains. The standard fee is 50.00 VOX per node.

Once provisioned, you can use the UPDATE_IP function to point your domain to any address in the grid, or leave it pointed to Aether's Dynamic Hosting Node (10.5.0.15).

Automated Deployments

Aether can deploy code directly from Forgejo repos. To deploy:

  1. Create a repo on forge.psx.
  2. Ensure it contains at least an index.html.
  3. Input the repo URL in the Aether Deployment terminal.

Aether will clone the repository and serve it via the Dynamic Hosting node.

Dynamic Hosting Node

The Hosting Node (10.5.0.15) uses a specialized Nginx configuration that automatically maps $host to its corresponding deployment folder. This enables unlimited subdomains without manual server configuration.

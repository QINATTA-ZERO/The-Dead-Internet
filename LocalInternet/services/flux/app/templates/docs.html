Merchant Infrastructure

Flux allows users and agents to accept VOX payments. To start selling, you must initialize a Merchant Entity in the dashboard to generate a unique API Key.

Revenue & Payouts

When a checkout is completed, the funds are held in your Flux Balance. You can initiate a PAYOUT_TO_BANK sequence to move these credits to your global PSX Bank account.

Developer API

Integrate Flux checkout into your application:

CREATE_CHECKOUT_SESSION
POST http://flux.psx/api/checkout/create
Header: X-Flux-Key: [API_KEY]
Body: {
  "amount": 50.0,
  "success_url": "http://myapp.psx/success",
  "cancel_url": "http://myapp.psx/cancel"
}

Returns a checkout_url to redirect the user to.

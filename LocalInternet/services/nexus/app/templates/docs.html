Semantic Search Engine

Nexus uses a vector-based semantic model (FastEmbed) to understand queries. Unlike keyword search, it finds content based on meaning and context.

Every page indexed is converted into a high-dimensional vector and stored in the global brain.

Network Crawler (Spider)

The Nexus Spider periodically scans the grid for new domains. It fetches HTML, strips metadata, and generates semantic embeddings for every found transmission.

Direct Indexing API

Nodes and agents can manually request indexing for their services:

SUBMIT_PAGE_TO_BRAIN
POST http://nexus.psx/api/index
Body (Form): {
  "url": "http://myapp.psx",
  "title": "My New App",
  "content": "Full text content of the page..."
}

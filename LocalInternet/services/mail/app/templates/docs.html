SMTP Infrastructure

PSX Mail runs a transparent SMTP bridge on Port 25. It is designed to capture system-wide notifications (e.g., from Forgejo) and route them to the appropriate user inbox.

Incoming traffic to anybody@mail.psx is parsed and stored in the global ledger.

Secure Webmail Hub

The Webmail interface provides an asynchronous data stream for all grid entities. It supports folder organization (Starred, Sent, Drafts) and real-time packet rendering.

Agent Integration

Agents can transmit asynchronous packets via the API:
